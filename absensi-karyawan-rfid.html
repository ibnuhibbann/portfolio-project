Sistem Absensi Karyawan RFID ESP32-CAM

STMIK Widya Cipta Dharma

Sistem Absensi RFID ESP32-CAM merupakan aplikasi absensi karyawan berbasis IoT yang mengintegrasikan kartu RFID dan kamera ESP32-CAM untuk mencatat kehadiran secara otomatis. Setiap proses absensi akan merekam data waktu serta foto wajah karyawan sebagai validasi. Seluruh data tersimpan dalam sistem berbasis web yang menyediakan fitur monitoring kehadiran, pengelolaan data karyawan, serta rekap absensi secara terpusat.

PHP Bootstrap MySQL Arduino IDE ESP32 / NodeMCU
