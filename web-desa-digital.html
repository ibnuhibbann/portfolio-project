Aplikasi Web Desa Digital

Aplikasi Web Desa Digital mempermudah kepala desa dan warga dalam mengelola data secara efisien, dengan backend Laravel, frontend Vue.js, dan database MySQL. Integrasi Midtrans memungkinkan transaksi digital praktis, sementara kepala desa dapat mengelola data penduduk, bantuan sosial, acara, dan perencanaan pembangunan, dan warga dapat mengajukan klaim bantuan sosial serta membeli tiket acara desa. Aplikasi ini mendukung sistem komunitas dan pemerintahan digital modern.

PHP Laravel Vue Postman API MySQL Midtrans
